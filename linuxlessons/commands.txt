Linux commands
ctrl+alt+T-открыть терминал
тильда-символ домашней директории
/-символ корневой директории
..-обозначает предидущую директорию
. -означает текущую директорию

  1. ls(list) - команда для вывода всех директорий(папка) и файлов в текущей(рабочей) директории в терминале
  
  2. cd (change/firectory) -  команда для изменения(переходов) одной лиректории не другую
          cd <название директории>
  3. mkdir (make directory) создает указанную директорию в рабочей директории в терминале
  4.  touch - создает указанный файл
  5. rm - удаление файла
  6. rmdir - удаление директории
  7. pwd - (present working directory) - показывает весь путь 
  8. rm -rf - принудительное удаление
  9. ls -a - выводит список как скрытых так и обычных файлов и дерикторий 
  10.nano <path_to_file> - откроется файл для редактирования в графичском интерфейсе
     *если нет файла, то nano попытается создать новвый файл
  11. cat <file_name> - выводит содержимое файла в терминал
